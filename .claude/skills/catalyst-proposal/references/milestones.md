# Milestone Structure

## Required Milestones by Budget

| Budget Range | Min Milestones | Total with Final |
|--------------|----------------|------------------|
| Up to 75k ADA | 2 | 3 |
| 75k - 150k ADA | 3 | 4 |
| 150k - 300k ADA | 4 | 5 |
| Over 300k ADA | 5 | 6 |

Final milestone always includes Project Close-out Report and Video.

## Milestone Components

### A. Title
- Clear, concise description
- Reflects deliverable outcome

### B. Outputs
- Deliverables or outcomes
- Specific, tangible results
- Avoid vague descriptions

### C. Acceptance Criteria
- Criteria for success
- Measurable conditions
- Verifiable by community

### D. Evidence of Completion
- Documentation type
- How to access/verify
- Specific artifacts

### E. Delivery Month
- Expected completion month
- Within project timeline
- Buffer for dependencies

### F. Cost in ADA
- Amount for this milestone
- Aligned with work scope
- All milestones sum to total budget

## Milestone Best Practices

**DO:**
- Make milestones specific and verifiable
- Include on-chain metrics where possible
- Allow buffer time for dependencies
- Front-load critical deliverables
- Include community testing periods
- Document acceptance criteria clearly

**DON'T:**
- Create generic milestones
- Bundle too many deliverables
- Back-load all work to final milestone
- Ignore dependencies
- Set unrealistic timelines
- Skip evidence specification

## Example Milestones

### Technical Project Example

**Milestone 1: Core Infrastructure**
- **Outputs**: Smart contract framework, test suite
- **Criteria**: Passes security review, 90%+ test coverage
- **Evidence**: GitHub repo, audit report
- **Month**: 2
- **Cost**: 40,000 ADA

**Milestone 2: Integration & Testing**
- **Outputs**: Mainnet deployment, API documentation
- **Criteria**: Live on testnet, 3+ integration partners testing
- **Evidence**: Testnet address, partner testimonials
- **Month**: 4
- **Cost**: 50,000 ADA

**Final Milestone: Launch & Documentation**
- **Outputs**: Mainnet launch, user docs, close-out report/video
- **Criteria**: 100+ transactions, docs published
- **Evidence**: Explorer link, documentation URL, PCR video
- **Month**: 6
- **Cost**: 30,000 ADA

## Budget Allocation Tips

- Development: 60-70%
- Testing/QA: 10-15%
- Documentation: 5-10%
- Marketing/Community: 5-10%
- Contingency: 5-10%
- Final Reporting: 5-10%
