# Proposal Structure

## 1. Proposal Setup

### Title (≤60 chars)
- Clear, unambiguous, concise
- Express what proposal is about
- Visible in voting app without opening

### Budget (ADA)
- Within category limits
- Must match milestone total

### Timeline (months)
- 2-12 months range
- Realistic for scope

### Problem Statement (≤200 chars)
- Well-defined core issue
- Displayed on voting app
- Hook that shows ecosystem pain

### Solution Overview (≤200 chars)
- Focus on what you WILL do
- Start with "We will build..." not "There should be..."
- Connect why and how

### Supporting Documentation
- Website, repository, portfolio links
- Whitepaper or publications
- Additional context for reviewers

### Dependencies
- List external dependencies
- Third-party software licenses
- Write "No dependencies" if none

## 2. Campaign Category Questions

### Innovation
- What makes idea new to Cardano?
- Unique approach vs existing solutions
- Clear improvements over funded work

### Prototype/MVP
- What will be demonstrated on-chain?
- How community can test it
- Access location/method

### Success Metrics
- Realistic blockchain usage projections
- Users, transactions, smart contract calls
- Tied to Cardano network metrics
- Data-based, not inflated

## 3. Project & Solution (Critical)

### SOLUTION (≤10k chars)
- How you perceive the problem
- Why this approach is optimal
- Who project engages
- How impact proven
- What's unique, who benefits, why matters to Cardano

### IMPACT (≤10k chars)
- Quantitative & qualitative measures
- Value proposition for community
- Measurement methodology
- Output sharing strategy

### CAPABILITY & FEASIBILITY (≤10k chars)
- Team's existing relevant capabilities
- Why team is best suited
- Fund management processes
- Feasibility demonstration steps
- Risk mitigation strategies

## 5. Final Pitch

### RESOURCES
- Team roster with LinkedIn profiles
- Roles and skills
- Recruitment plans if needed
- Engagement evidence (Discord, Telegram)

### BUDGET & COSTS
- Itemized breakdown
- Third-party products/services
- Marketing, project management, documentation
- Reference Fund Rules for eligibility

### VALUE FOR MONEY
- Context for costs
- Supporting evidence for rates
- Comparison to average wages/freelance rates
