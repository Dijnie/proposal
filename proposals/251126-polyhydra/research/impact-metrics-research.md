## 3. On-Chain Metrics Methodology

### Hydra Transaction Tracking
```
Hydra Head Logs
  └── Opening transactions (L1)
  └── In-head transactions (L2)
  └── Settlement transactions (L1)
```

### Wallet Activity
```
CIP-30 Connections
  └── Unique addresses
  └── Connection frequency
  └── Transaction signatures
```

### NFT/Token Distribution
```
Minting Policy ID
  └── NFTs minted
  └── Unique recipients
  └── Secondary sales (DEX)
```
